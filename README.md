## Commands to start the project

### Install dependencies
```bash
npm install
```

### Start the project

if you have the node 22 installed, you can run the following command:
```bash
node --run dev
```
or use your preferred package manager:
```bash
npm run dev
```

### Run de json server
```bash
cd server
json-server --watch db.json
```
